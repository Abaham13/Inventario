# Inventario
lo que se desea realizar es un sistema de inventario con interfaces gráficas donde el usuario pueda agregar, modificar y eliminar tanto productos como almacenes. Todo esto debe estar controlado por diferentes roles de usuario, para que no cualquiera pueda hacer cambios importantes.
